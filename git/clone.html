克隆库 ​

下面就要将你在github上创建的仓库克隆到你的本地来，方便以后进行代码上传

打开你所创建的仓库：（后面会用到箭头指的网站

可以现在本地磁盘里建一个文件夹（我是在e盘建了github文件夹）

然后再打开Git Bash ，切换到刚建的文件夹位置 cd /e/github

然后再克隆一个仓库，输入 git clone 网址 （网址是上面图片里的）

然后再回到之前在e盘github文件夹里面出现一个文件夹

可以现在yuchaifan1.git.io文件夹里建一个txt文本，后面可以明显看到上传内容

然后再此文件夹右键打开Git Bash 并输入 git add . 回车（将打包的文件放到暂存区 上传全部文件）

然后在输入：git commit -m “描述内容，备注” ,输入之后出现以上情况 提交代码到本地仓库

然后输入git pull 同步远程最新代码到本地，并自动合并

在输入git push 提交 将本地已提交的代码推送到远程仓库，共享给团队

成功之后，再回到自己github库里面可看到自己提交的文本
