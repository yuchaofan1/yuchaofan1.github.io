node.js ​

官网下载

win+r打开命令行

检查是否下载成功 node -v

                        npm -v

打开node.js文件

创建文件 node_global 和 node_cache

cmd命令里的路径不可以有空格

路径 地址要点到需要复制的文件里面

npm config set prefix "路径\node_global"

npm config set cache "路径\node_cache"



配置环境变量 ​


环境变量--新建--NODE_PATH

地址：路径\node_global\node_modules

用户变量——Path(npm编辑)——路径\node_global

系统变量——Path——新建——%NODE_PATH%

一直确定就可以了

win+R——npm install express -g 或 npm config list
